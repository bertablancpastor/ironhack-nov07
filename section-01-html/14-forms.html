Form Elements Syntax

form: elemento padre para inicializar un formulario.

label for: son clicables y lo relaciona con el input

input: son los elementos mas usados y hay de muchos tipos

select: son para selecionar una opcion

text area: espacio donde el usuario puede escribir en el formulario

button: define un click para el boton
