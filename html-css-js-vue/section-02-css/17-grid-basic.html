QUE ES GRID?

Da la capacidad de distribuir el contenido con cuadriculas, herramienta potente para distribuir la información. Por defecto el GRID se debe dividir en 12 COLUMNAS y 12 FILAS.

GRID CONTAINER

El "container"

GRID ITEM

Son los desendientes directos del GRID CONTAINER

GRID LINE

Las linias divisorias de columna o fila

GRID CELL

Espacio de la cuadricula, el intetrior de la columna y la fila

GRID TRACK

El espacio entre toda la linia o columna

GRID AREA

El total rodeado por 4 linias

FRACCIONES

Unidad de valor que fraccion por partes iguales

AREA

Define la plantilla de la cuadricula haciendo referencia a los nombres de las areas

GRID TEMPLATE

Es un shorthand, primero pide establacer las filas y despues las columnas

COLUM-GAP /ROW-GAP

Para hacer el margen entre las casillas se puede usar GAP con COLUM y ROW

Para hacer el shorthand solo se pondra GAP. Con dos medidas diferentes primero es row y el segundo es bottom.

GRIT ITEMS

Span: palabra reservada para decir que des de donde se inicia añade el numero que le indiques
